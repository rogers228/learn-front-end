# manifest.json

1. PWA 漸進式 Web 應用程式 需要一個manifest.json(PWA應用程序資訊檔)
2. 通常存放與index.html同位置
3. PWA應用程序資訊檔 manifest.json 他是一個通用標準，用來定義PWA的資訊，可以在各大行動裝置瀏覽器適用，內容例如安裝圖示

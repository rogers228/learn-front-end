# 單文件組件
Single-File Component簡稱SFC
是將邏輯JS 架構輸入HTML 樣式CSS封裝為一個獨立檔案，並且可以重用
一個檔案就是一個組件component，副檔名是.vue
跟react, svelte最新的前端框架皆是如此
它採用標籤區隔語法
```vue
<template> <script> <style>
```

# 與SFC相對的就是傳統的

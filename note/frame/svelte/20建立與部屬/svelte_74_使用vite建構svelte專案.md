# 說明

vite是一個開發服務器
使用vite開發服務器時，不需要時時編譯svelte就能執行。

## 安裝
```
cd /d your_path  移動到目錄
npm create vite@latest home -- --template svelte   建立home資料夾為基於vite的svelte app

cd home  移動到homm
npm install  安裝 npm
```

## 啟動開發服務器
```
npm run dev   開啟開發vite服務器 不需要時時編譯svelte就能執行。
```

## 將svelte編譯為js
```
npm run build  編譯至dist資料夾
```

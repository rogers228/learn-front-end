# title

https://bulma.io/documentation/elements/title/

.title  標題
.subtitle  次標題

## size 設定
.is-1 ~ .is-6
.is-3 預設值
.is-spaced 周圍需要空間

不使用 is-size-1 控制

## 控制
.has-text-centered  置中
.has-text-*
.is-inline-block   抑制後方元素換行

## 抑制block換行
block 轉 inline
block element 會換行，例如 h1-h6
加上 is-inline-block 轉換為 inline-block

```
border-style:
none：沒有邊框。
solid：實線邊框。
dashed：虛線邊框（破折號樣式）。
dotted：點線邊框。
double：雙線邊框。
groove：凹槽邊框，看起來像是3D的凹陷效果。
ridge：脊狀邊框，看起來像是3D的凸起效果。
inset：嵌入邊框，看起來像是元素嵌入畫面中。
outset：突起邊框，看起來像是元素從畫面中凸出來。
hidden：隱藏邊框，和 none 類似，但主要用於解決表格邊框衝突問題。
```

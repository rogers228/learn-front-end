# 箭頭函數

```js
    (n) => {        //()括號為匿名函數(不需要函數名稱)
                    //()括號內為指名引數(引數名稱可以自訂名稱，爾後表達式內可引用)
                    // => 箭頭
                    //{}花括號內為表達式

        return n+1  // return 回傳值
    };
```


## 單行箭頭函數
```js
    n => n+1     // 等價上方
                 // 省略匿名函數的()括號, {}花括號, return

```

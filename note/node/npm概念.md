# npm
pm是Node的套件管理器，如同pip是python的套件管理器


# npm init
初始化你的專案(建立json記錄檔)
其實npm init就是建立一個專用的資料夾

```
cd /d 你的目錄位置
npm init
```
用NPM 來創建的 Project 的 json檔案

```
npm init -y
```
跳過選項直接建立


# npm install 
安裝套件，會將套件及其依賴，安裝至專案資料夾，缺點檔案無法共用，但相對簡單乾淨隔離
部屬時可依照json檔直接安裝

```
npm install modname
```
